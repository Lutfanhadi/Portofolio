About me

Curious about me? Here you have it:

Saya adalah seorang mahasiswa di STMIK IKMI Cirebon yang memiliki minat besar dalam pengembangan antarmuka pengguna (UI) dan pengalaman pengguna (UX) untuk aplikasi web. Dengan fokus pada Frontend Development, saya terus membangun keahlian dalam pemrograman, desain responsif, dan optimalisasi performa antarmuka.

Saat ini, saya mendalami pengembangan aplikasi berbasis web menggunakan teknologi seperti HTML, CSS, JavaScript, Tailwind CSS, dan kerangka kerja modern lainnya. Saya juga memiliki kemampuan dalam mengintegrasikan antarmuka dengan backend untuk menciptakan aplikasi yang fungsional dan menarik.

Sebagai mahasiswa yang proaktif, saya selalu mencari peluang untuk belajar, berinovasi, dan berkontribusi pada proyek-proyek yang dapat memberikan pengalaman nyata dalam dunia teknologi informasi. Saya terbuka untuk kolaborasi, magang, atau peluang kerja sama lainnya di bidang pengembangan frontend.

Mari terhubung untuk berbagi wawasan dan menciptakan inovasi bersama!
